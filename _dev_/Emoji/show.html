Pour produire une image d'émoji :

  1. Copier-coller l'émoji dans ce fichier
  2. Recharger ce fichier pour le voir.
  3. Faire une copie partielle d'écran avec CMD-MAJ-4.
  4. Récupérer le fichier dans le dossier captures courant (DD MacOsCatalina/Captures).
  5. Ouvrir le fichier avec Gimp.
  6. Ne pas utiliser la transparence, faire une sélection de la couleur blanche et “Édition/Agrandir…” la sélection de 2 pixels.
  7. Effacer avec Fn-Effacement arrière.
  8. Améliorer en gommant les blancs restants.
  9. Mettre l'image dans un cadre de 480px/480px (Crop - Maj-C).
  10. Exporter l'image en png dans le dossier ./img/emojis/ du site de l'atelier.
  11. Pour faire une version ombrée, cliquer sur le bouton “Ombrer” ci-dessous et recommencer les opérations précédentes en nommant le fichier final comme le fichier précédent et “-shadowed.jp2.
  12. Dans le fichier __first/constants/emojis.rb définir une constante pour cet emoji en s'inspirant des autres.
